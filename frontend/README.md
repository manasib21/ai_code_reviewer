# AI Code Review Tool - Frontend

React + TypeScript frontend for the AI Code Review Tool.

## Features

- Modern, responsive UI
- Code review interface
- Review history
- Configuration management
- API usage audit dashboard
- HTML report download

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Technologies

- React 18
- TypeScript
- Vite
- React Router
- Axios
- React Syntax Highlighter
- Lucide React Icons
