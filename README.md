# AI Code Review Tool

A comprehensive AI-powered code review tool that detects bugs, security vulnerabilities, code style issues, and documentation problems across all programming languages.

## Features

- **Multi-Model AI Support**: OpenAI GPT and Anthropic Sonnet
- **Comprehensive Detection**: Bugs, security, style, and documentation
- **Multiple Interfaces**: Web application and IDE extension
- **Version Control Integration**: Git hooks, CI/CD, and PR/MR comments
- **Flexible Configuration**: Custom rules, severity levels, ignore patterns
- **Advanced Features**: Code suggestions, learning mode, history tracking, collaboration, API usage audit

## Project Structure

```
├── backend/          # FastAPI backend server
├── frontend/         # React web application
├── extension/        # VS Code extension
├── hooks/           # Git hooks scripts
├── ci-cd/           # CI/CD integration scripts
└── config/          # Configuration files and templates
```
